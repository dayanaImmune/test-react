# Veterinaria App Pro

Proyecto de práctica para testing con React + Vitest + React Testing Library.

## Qué incluye
- Formulario completo de pacientes
- Validaciones
- Búsqueda por nombre
- Filtro por tipo
- Filtro de urgentes
- Marcar paciente como atendido
- Eliminar paciente
- Contador de pacientes
- Más de 20 tests

## Instalación
```bash
npm install
npm run dev
npm run test
```

## Estructura
- `src/components`: componentes
- `src/tests`: tests unitarios e integración

## Nota
Aunque aquí se usa **Vitest**, la sintaxis es prácticamente la misma que usarías con Jest para este tipo de pruebas.
